Contribute to the site: prose.io for a slick editor, or git commands:

$ git clone https://github.com/area3001/area3001.github.io.git
      $ 							# Modify, add files
      $ git status    					# Check what you did
      $ git add add_or_changed_files			# Add the files (stage)
      $ git commit -m "description of your changes" 	# Commit the files local
      $ git push origin master				# Sync/upload to the GitHub server
Monks without push permission, should fork the project, and send a pull request.
